# 连连看_QtRefresh

一个基于 **Qt 6** 重构的连连看游戏项目，支持经典模式、休闲模式与计时模式，包含提示、重排、自动无解重排、路径连线显示、音频设置等功能。

> 本项目由历史 **MFC 连连看（LLK）项目迁移** 而来，在保留核心玩法规则的基础上完成了 Qt 化重构与界面刷新。

---

## 项目特性

- ✅ **三种游戏模式**
  - 经典模式（9×14）
  - 休闲模式（8×8）
  - 计时模式（8×12，含倒计时与结算）

- ✅ **连连看核心规则**
  - 两张相同图案可消除
  - 连线最多允许 **2 个拐点**
  - 支持“绕棋盘外圈”连接

- ✅ **智能玩法辅助**
  - 提示功能：高亮一对可消除图案
  - 重排功能：打乱当前未消除图案
  - 无解检测：自动判断当前局面是否可解
  - 自动重排：无解时自动重排直到可解（可配置）

- ✅ **可视化体验**
  - 消除时绘制连线路径（含拐点路径）
  - 不同模式的差异化主题按钮与帮助文案
  - 背景图与图标资源统一管理

- ✅ **音频与设置持久化**
  - 背景音乐播放/静音/音量调节
  - `QSettings` 保存用户音量、静音和自动重排设置

## 技术实现概览

### 1. 分层结构

- **UI 层**
  - `MainWindow`：主菜单与模式入口
  - `GameWindow`：游戏窗口基类（棋盘创建、点击交互、路径绘制、提示样式）
  - `ClassicWindow / RelaxWindow / TimedWindow`：模式差异化实现

- **核心逻辑层**
  - `GameLogic`：地图生成、连通判定、路径搜索、提示与可解性检测

- **功能组件层**
  - `AudioManager`：背景音乐与音频状态管理（单例）
  - `SettingsDialog / HelpDialog / ResultDialog`：设置、帮助、结算弹窗

### 2. 核心算法说明（GameLogic）

- 地图生成采用“**成对数字 + 随机打乱**”方式，确保可进行配对消除。
- 连通判定采用 **BFS（广度优先）状态搜索**：
  - 状态包含坐标、方向、拐点数
  - 限制最多 2 个拐点
  - 通过扩展边界实现“外圈绕行”规则
- 路径输出版本会回溯前驱节点，提取“起点-拐点-终点”关键路径用于 UI 画线。

### 3. 计时模式机制

- 基于 `QTimer` 每秒刷新倒计时
- 支持暂停/继续（输入禁用与遮罩显示）
- 结束后弹出结算对话框，显示用时、剩余时间、消除对数与评分

## 资源文件说明

本项目的静态资源主要位于以下目录：

- `audio/`：音频资源（如背景音乐 `bgm.mp3`）
- `images/`：图片资源（背景图、图标、牌面图案等）

> 运行前请确保资源文件已正确加入 Qt 资源系统（`.qrc`）并可通过 `:/audio/...`、`:/images/...` 路径访问。

## 运行环境

- Qt 6.11（建议6.0以上）
- C++17（或与项目配置一致的标准）
- Windows 10及以上版本
- 支持 Qt Widgets 与 Multimedia 模块

## 项目来源与说明

本项目为对 **MFC 版本连连看老项目** 的继承与重构版本（Qt Refresh），目标是：

1. 保留原有玩法规则与核心体验  
2. 提升界面表现与交互一致性  
3. 统一资源管理与配置持久化方案  
4. 为后续拓展（排行榜、更多模式、动画优化等）提供清晰结构基础
